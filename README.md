# pydomain

A Python 3.12+ library implementing Domain-Driven Design (DDD), Command-Query Responsibility Segregation (CQRS), and Event Sourcing (ES) building blocks.

## Installation

### From source (development)

```bash
git clone https://github.com/mgourlis/pydomain.git
cd pydomain
uv sync --extra dev
```
