# How-to Guides

Task-oriented documentation — solve a specific problem with copy-paste-ready steps.

Each subfolder mirrors the concepts structure. Pick a task, follow the steps, get it done.

## Sections

| Section | Tasks | Adoption Level |
|---------|-------|---------------|
| [DDD](ddd/) | Define entities, value objects, aggregates, events, repositories | Level 1 |
| [CQRS](cqrs/) | Define commands, queries, handlers, configure buses | Level 2–3 |
| [Event Sourcing](event-sourcing/) | Event-sourced aggregates, projections, upcasting | Level 4–5 |
| [Sagas](sagas/) | Define sagas, compensation, error handling | Level 5 |
| [Infrastructure](infrastructure/) | Bootstrap, register handlers, configure message broker | Level 3+ |
| [Testing](testing/) | Use fakes for repositories, UoW, event stores | All levels |
| [Recipes](recipes/) | End-to-end integration patterns | All levels |
